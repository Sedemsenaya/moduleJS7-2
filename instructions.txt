
=================
MODULE 2 LAB DESCRIPTION
=================
In this lab, you will use jQuery to perform various DOM
related operations. Developing web applications requires
you to fully manipulate the HTML using JavaScript/jQuery.
This includes handling events, reading form and HTML elements,
creating/removing HTML elements, etc. These exercises will
help you refine your front-end DOM manipulation skills using
jQuery.
